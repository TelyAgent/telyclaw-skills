# Topical Template

Identifies common topics across messages and groups them. Best for cross-channel analysis.

## Output Format

```markdown
# {Channel names} · Topical Briefing
**Time range:** {start date} ~ {end date}

---

## Trending Topics

### {Topic 1 — concise 3-6 word title}
- [{Channel name}] {related message summary}
- [{Channel name}] {related message summary}

### {Topic 2}
- [{Channel name}] {related message summary}

---

## Other
- [{Channel name}] {noteworthy but ungrouped}
...

---

N messages across T topics
```

## Rules

- After reading all messages, identify 2-5 recurring themes.
- Assign each message to the most relevant topic; don't duplicate.
- Messages that don't fit any topic but are still valuable go under "Other".
- Topic names should be specific (e.g. "NVIDIA earnings" not "Finance").
