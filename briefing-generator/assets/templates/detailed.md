# Detailed Summary Template

Preserves key details from original messages. Best for in-depth reading.

## Output Format

```markdown
# {Channel names} · Detailed Briefing
**Time range:** {start date} ~ {end date} · **N messages**

---

## {Channel 1 name}

### {YYYY-MM-DD}

**{First sentence as headline}**
{Full or excerpted content — keep data, figures, conclusions}
> {views} views · {forwards} forwards

**{First sentence as headline}**
{Full or excerpted content}
> {views} views · {forwards} forwards

---

## {Channel 2 name}
...
```

## Rules

- Preserve data, numbers, and conclusions from the original messages.
- For long messages, excerpt the core paragraphs. Mark truncation with "… (excerpted)".
- Group by date, ordered by time within each day.
- Include view and forward stats per message.
